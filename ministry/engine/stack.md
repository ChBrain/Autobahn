# REMEMBER Stack

## Owner
Project: REMEMBER

## Overview
A content module for the Autobahn world documenting federal transport minister tenures and infrastructure decisions.

## Module Architecture
REMEMBER inherits the Autobahn engine (instructions.md, positions, pieces, processes). It adds:
- 28 personas (federal ministers 1949-2025)
- 27 pieces (infrastructure decisions, policies, networks)
- 7 positions (ministerial roles, offices, mandates)
- 5 places (ministry locations, institutional centers)
- 1 master process (Govern Traffic, subdivided into Take Office, Run Office, Leave Office)

## Deployment
Deployable as standalone bundle: `remember.zip` and `remember.pdf`

Manifest: [ministry.md](../../ministry.md)

## Versioning
Inherits Autobahn versioning scheme. Individual file headers show tenure dates and policy scope.

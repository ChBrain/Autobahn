# REMEMBER

The REMEMBER module for the Autobahn world. Manifest for the remember-bundle.

This file is the manifest the deployer reads when building `dist/remember.zip`.
The bundle is exactly the files linked from this file: the REMEMBER content plus
the Autobahn engine frame that REMEMBER inherits.

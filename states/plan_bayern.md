# Autobahn Build Plan: Bayern

*v1.0 — 03.05.2026*

## Status: STUB

Holds empty. Bayern has the second-largest Autobahn network in Germany,
the highest proportion of unrestricted sections, and the Alps on the horizon
of the southern roads.

## Autobahnen

A3, A6, A7, A8, A9, A92, A93, A94, A95, A96, A99

## Phases

### Phase 0 — State file update (now)
Define the Alpine character. Speed limit politics. The unrestricted sign as the dominant visual.
Files: 1 updated.

### Phase 1 — A9 Bayern (3 sessions)
Munich to northern border (Nuremberg-Berlin corridor). The prestige north-south route.
Files: 1 road file + 20-25 exit files.

### Phase 2 — A8 Bayern (3 sessions)
Munich to Salzburg, Munich to Stuttgart. The Alpine approach — the Alps appear on the horizon.
Files: 1 road file + 18-22 exit files.

### Phase 3 — A99 Muenchner Ring (2-3 sessions)
The ring around Munich. The city that cannot be reached, only circled.
Files: 1 road file + 15-20 exit files.

### Phase 4 — A3 Bayern (2-3 sessions)
Frankfurt to Passau. The Danube crossing. Austrian border.
Files: 1 road file + 15-18 exit files.

### Phase 5 — A7 Bayern (3 sessions)
Wurzburg south to Austrian border at Kufstein. The longest single road in Germany through its longest state section.
Files: 1 road file + 20-25 exit files.

### Phase 6 — A92, A93, A94, A95, A96 (3-4 sessions)
Regional and Alpine roads. Inntal, Innkreis, Rosenheim, Lindau corridors.
Files: 5 road files + 25-30 exit files.

### Phase 7 — A6 Bayern (1-2 sessions)
Nuremberg to Czech border. Relatively sparse.
Files: 1 road file + 8-10 exit files.

### Phase 8 — State file final (1 session)
Full linked Holds, all 11 roads.
Files: 1 updated.

## Totals

~120-150 new files. 18-22 sessions.

## Defining Withheld

The Alps: a full wall of white on the horizon from A8, A95, A96 on clear days.
Present. Not reachable from the carriageway.
Neuschwanstein: 10 km from the right exit, absent from any road view.
The beer garden: structurally inaccessible from the Autobahn network.
The Autobahnkirche: a small chapel at the roadside, present without explanation.

# Autobahn

*A motorway world built from the journeys that happen on the same road without knowing about each other.*

---

A world built from the particular of a motorway journey.

The Kilometerstein counts every hundred metres. It does not recalculate. The road does not know which journey it is on. It is always on one.

The world holds the journeys that happen in parallel on the same road without knowing about each other. Each persona carries something that began before the Auffahrt and will not resolve before the Abfahrt.

Direction is in the Driver, not the place. The road goes both ways. The world picks the persona and place with the strongest pull on what you bring.

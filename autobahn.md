# Autobahn

The full Autobahn world. Manifest for the all-bundle.

This file is the manifest the deployer reads when building `dist/autobahn.zip`.
The bundle is exactly the files linked from this file. Add a road, a Bundesland,
or a frame piece to the world, then link it here for it to ship in the
all-bundle.

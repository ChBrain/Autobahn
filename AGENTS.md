# Autobahn Agents

This file registers custom agents for the Autobahn project.

## Available Agents

- [autobahn-release](./.github/agents/autobahn-release.agent.md) - Handles version releases for Autobahn with safety gates
- [minister-transport](./.github/agents/minister-transport.agent.md) - Federal Minister for Transport - dynamically selects minister persona based on tenure context (BRD 1949-, DDR 1949-1990)
